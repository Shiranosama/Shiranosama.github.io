## 💻 安裝

### Git 安裝

在博客根目錄裡安裝穩

```shell
git clone -b theme https://github.com/Shirano39/Shirano39.github.io.git themes/Arashi
```

## ⚙ 應用主題

修改hexo配置文件`_config.yml`，把主題改為`Arashi`

```
theme: Arashi
```

如果你沒有pug以及stylus的渲染器，請下載安裝

```shell
npm install hexo-renderer-pug hexo-renderer-stylus --save
```
